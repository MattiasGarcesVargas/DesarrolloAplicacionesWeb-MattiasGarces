# 📖 Descripción

**Essence** es una página web estática enfocada en la promoción y venta de perfumes originales, inspirados y decants.

El proyecto busca ofrecer una experiencia visual premium y organizada para usuarios que desean explorar fragancias, comparar precios y realizar pedidos de forma rápida desde cualquier dispositivo.

La propuesta está dirigida principalmente a usuarios de **Guayaquil y Samborondón**, permitiendo navegar por un catálogo elegante y simular compras mediante un carrito interactivo.

<br>

<hr>

<br>

# 🎯 Objetivo del Proyecto

Desarrollar una tienda web moderna y responsive que permita:

* Explorar un catálogo de perfumes.
* Visualizar información relevante de cada fragancia.
* Agregar productos a un carrito de compras.
* Mejorar la experiencia de búsqueda y selección.
* Facilitar el contacto mediante WhatsApp y redes sociales.
